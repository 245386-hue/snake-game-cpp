# Snake Game in C++

A console-based Snake Game developed using C++ and Windows console functions.

## Features
- Real-time snake movement
- Arrow key controls
- Score tracking system
- Collision detection
- Dynamic speed increase
- Restart and quit options

## Technologies Used
- C++
- Windows Console API
- Object positioning and game logic

## Concepts Practiced
- Arrays
- Loops
- Functions
- Enums
- Game logic
- Keyboard handling

## Controls
- Arrow Keys → Move Snake
- ESC / X → Quit Game
